### Function Declartion : We can declare function in 3 ways: -->

        1. Simple way :
                function Greet(){
                    console.log("Hello World");
                }


        2. Giving variable to a function :
                let Greet = function (){
                    console.log("Hello World");
                }


        3. Using Arrow Function
                var Greet = () => {
                    console.log("Hello World");
                }
